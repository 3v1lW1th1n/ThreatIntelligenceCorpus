
Access Kali Linux Remotely on Windows System through Putty

First Open Your Kali Linux and Follow this path
Applications->Kali Linux->System Services->SSH->sshd start

Download Putty for Windows
Run Putty and put in the IP address for your Kali System. You can get this by typing “ifconfig”
My IP address was 192.168.1.120 in this case. Also, make sure port 22 is entered and select SSH as shown below:

Now Click “Open”
